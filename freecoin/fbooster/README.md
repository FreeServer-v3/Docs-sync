# 🚀 FBooster

FBooster，即參考 [Hypixel 的 Network Booster](https://hypixel.net/threads/guide-everything-about-network-boosters.2465740/)，是一種能夠加倍獲取 FreeCoin 的加速器。

簡單來說，FBooster 作用時間內，所有人都可以在各種獎勵中獲得倍數加成，並且部分加成會回饋至 FBooster 持有者。
