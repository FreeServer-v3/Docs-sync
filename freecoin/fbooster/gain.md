# 獲取方式

FBooster 目前有以下獲取方式:

## Discord 伺服器加成

點擊 伺服器加成 > 加成此伺服器:

{% hint style="info" %}
加成需要人工審核。在加成後您需要透過FS信箱聯絡我們，告知後我們將發放 FBooster 給您。
{% endhint %}

透過加成我們的 Discord 伺服器，你可以得到:

* 一個 12小時 1.5倍 FBooster
* 一個 24小時 1.2倍 FBooster
