# 🔸 管理面板 (dash)

進入管理面板後，您應該會看到...

<figure><img src="../../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

相關功能、資訊分別如下：

1. 剩餘 FreeCoin，顯示您目前可供使用的虛擬代幣。
2. 左側功能列，可供您進入各分頁。
3. 可用資源列，顯示您可使用的資源與數值。
4. 伺服器列表，顯示您所擁有的伺服器與分配之資源。
5. 帳戶資訊，顯示您在面板中 Username 與 Email，還有面板連結。
